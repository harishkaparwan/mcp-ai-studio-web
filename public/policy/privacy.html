Privacy Policy

Last updated: June 2025

Overview

MCP AI Studio is a client-side tool. It does not collect, store, or transmit any personal data to any server operated by the developer.

Data Storage

All configuration (MCP server URLs, client names, titles) is stored exclusively in your browser's localStorage or VS Code / Windsurf workspace settings. This data never leaves your device.

Network Requests

The extension and web app make requests only to MCP servers that you explicitly configure. No telemetry, analytics, or crash-reporting requests are made.

Third-Party Services

This page is hosted on GitHub Pages. GitHub may collect standard server access logs (IP address, browser user-agent, referrer). Refer to GitHub's Privacy Statement for details.

Contact

Questions? Open an issue on GitHub.
